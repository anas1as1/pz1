# Revo Coffee

Учебный проект по дисциплине «Веб-разработка».

Сайт выполнен на чистом HTML с использованием семантической вёрстки.
Структура страницы соответствует макету. В проекте используются
теги `header`, `main`, `section`, `article`, `nav`, `footer`.

## Структура проекта
- `index.html` — главная страница
- `assets/img` — изображения
- `assets/icons` — иконки

## Используемые технологии
- HTML5
